# LavaRapidu

Sistema de gerenciamento para um lava-rápido desenvolvido em Java Web utilizando JSP, Servlets, MySQL e padrões de projeto.

## Tecnologias Utilizadas

* Java
* JSP
* Servlets
* MySQL
* Apache Tomcat
* JDBC
* HTML
* CSS

## Padrões Implementados

### MVC

Separação entre Model, View e Controller.

### DAO

Responsável pela persistência dos dados.

### Builder

Utilizado para construção de objetos complexos.

### Decorator

Permite adicionar serviços extras ao atendimento sem alterar a classe principal.

### Command

Encapsula operações em objetos de comando.

## Princípios SOLID

* SRP (Single Responsibility Principle)
* LSP (Liskov Substitution Principle)
* DIP (Dependency Inversion Principle)

## Funcionalidades

* Cadastro de Clientes
* Cadastro de Veículos
* Cadastro de Serviços
* Cadastro de Atendimentos
* Listagem de Registros
* Exclusão de Registros

## Banco de Dados

O banco utilizado é MySQL.

Script disponível na pasta do projeto.
